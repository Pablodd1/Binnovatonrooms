## 2026-06-25 - Improve accessibility of login form error states\n**Learning:** The run_in_bash_session tool truncates command output at exactly 1000 characters. When inspecting files for accessibility improvements, using 'cat' on the entire file can miss crucial elements at the end, leading to hallucinated or groundedness-violating execution plans. \n**Action:** Use targeted commands like 'tail -n', 'head -n', 'sed -n', or 'grep -C' instead of 'cat' to read specific, relevant sections of files reliably when performing Palette UX tasks.
