## 2026-06-24 - Typed Arrays for Image Processing\n**Learning:** Client-side image processing (like camera frame scoring) is highly sensitive to Garbage Collection (GC) overhead when generating temporary dynamic arrays.\n**Action:** Use pre-allocated typed arrays (e.g. `Float32Array`) and basic `for` loops instead of pushing into dynamic arrays and running expensive array methods (`reduce`, `filter`) in high-frequency paths.
